# Idioma

O Delfus fala **português (pt-BR)** por padrão e também suporta **inglês (en-US)**.

## Trocar o idioma

Use o comando de idioma no servidor:

```
/idioma
```

A escolha vale para o servidor inteiro e é lembrada pelo bot.

!!! note "Por que pt-BR por padrão?"
    O Delfus assume português por padrão. O Discord só expõe um idioma confiável do
    servidor quando o **Modo Comunidade** está ativado — fora disso, o valor não é
    confiável, então o bot mantém o português. Se preferir inglês, basta usar `/idioma`.

## Pré-visualizar

Na mensagem de boas-vindas enviada quando o bot entra, há um menu **🌐 Ver em outros
idiomas** que mostra o conteúdo em cada idioma disponível antes de você decidir.
