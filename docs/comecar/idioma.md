# Idioma

O Delfus fala português (pt-BR) por padrão e também suporta inglês (en-US).

## Trocar o idioma

Use o comando de idioma no servidor:

```
/idioma
```

A escolha vale para o servidor inteiro e o bot a lembra.

!!! note "Por que pt-BR por padrão?"
    O Discord só expõe um idioma confiável do servidor quando o Modo Comunidade está ativado. Fora disso o valor não é confiável, então o bot mantém o português. Se preferir inglês, use `/idioma`.

## Pré-visualizar

A mensagem de boas-vindas enviada quando o bot entra traz um menu 🌐 Ver em outros idiomas. Ele mostra o conteúdo em cada idioma disponível antes de você decidir.
