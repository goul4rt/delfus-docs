# Adicionar o bot

Convide o Delfus para o seu servidor Discord e dê as permissões certas. Em poucos minutos ele já está rodando.

## 1. Convidar o Delfus

Abra o link de convite e escolha o servidor. Você precisa ter a permissão **Gerenciar Servidor** nesse servidor.

[Convidar o Delfus](https://admin.delfus.app){ .md-button .md-button--primary }

!!! tip "Permissões"
    Mantenha as permissões sugeridas no convite. Vários módulos dependem delas: moderação usa **Banir Membros**, cargos usa **Gerenciar Cargos**, boas-vindas usa **Gerenciar Mensagens**. Sem a permissão, o módulo não funciona.

## 2. Mensagem de boas-vindas

Quando o Delfus entra, ele manda uma mensagem no servidor e uma DM para o dono, com um resumo dos próximos passos. Tudo já vem em **português** por padrão. Para mudar, veja [Idioma](idioma.md).
