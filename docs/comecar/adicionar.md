# Adicionar o bot

Para usar o Delfus, convide-o para o seu servidor e garanta que ele tenha as
permissões necessárias.

## 1. Convidar o Delfus

Use o link oficial de convite e selecione o servidor onde você quer adicionar o bot.
Você precisa ter a permissão **Gerenciar Servidor** no servidor escolhido.

[Convidar o Delfus](https://admin.delfus.app){ .md-button .md-button--primary }

!!! tip "Permissões"
    Mantenha as permissões sugeridas no convite. Vários módulos (moderação, cargos,
    boas-vindas) dependem de permissões como **Gerenciar Cargos**, **Banir Membros**
    e **Gerenciar Mensagens**. Sem elas, o módulo correspondente não funciona.

## 2. Mensagem de boas-vindas

Assim que o Delfus entra, ele envia uma mensagem de boas-vindas no servidor e uma
DM para o dono, com um resumo do que fazer a seguir. A mensagem já vem em
**português** por padrão — veja [Idioma](idioma.md) para mudar.
