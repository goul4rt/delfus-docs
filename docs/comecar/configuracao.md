# Configuração inicial

Você configura o Delfus pelo **Dashboard**, o painel web onde ativa módulos e escolhe canais e cargos do seu servidor Discord.

[Abrir o Dashboard](https://admin.delfus.app){ .md-button .md-button--primary }

## Como funciona

1. Acesse o [Dashboard](https://admin.delfus.app) e faça login com o Discord.
2. Selecione o servidor onde o Delfus está.
3. Ative e configure cada módulo na barra lateral (moderação, boas-vindas, VIP, etc).

As mudanças feitas no painel são aplicadas no bot automaticamente.

## Comandos no Discord

Você também pode usar comandos de barra no servidor. Digite `/` no campo de mensagem para ver todos os comandos, ou use `/help` para uma visão geral.

!!! info "Painel ou comandos?"
    Use o **Dashboard** para configurar (ativar módulos, definir canais e cargos) e os **comandos** para o dia a dia (aplicar punições, consultar dados, etc).
