# Delfus

**Análise avançada, moderação e gestão para comunidades Discord.**

O Delfus é um bot completo para o seu servidor: moderação e anti-raid, programa VIP,
boas-vindas, aniversários, starboard, cargos por reação e um painel web para configurar
tudo visualmente — de graça.

[Adicionar ao meu servidor](comecar/adicionar.md){ .md-button .md-button--primary }
[Abrir o Dashboard](https://admin.delfus.app){ .md-button }
